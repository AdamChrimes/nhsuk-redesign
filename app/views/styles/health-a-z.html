What it is

The Health A-Z is a directory page on the NHS website where people can look for conditions and sympotoms. It comprises an alphabetical list of anchor links, which jump the user to the relevant letter.

User needs

As someone with a symptom or condition that I can name or describe, I NEED to be able to look for the right link, SO THAT I find the correct information and advice.

What was being used

The previous iteration featured an alphabetical index which confused mobile phone users, leading them to think it was a phone's keyboard, rather than a series of individual text links.

Iterations

Research

During a research lab in 2018:

All users were able to use the A-Z list, and this behaved as they expected.

However, some users missed the quick anchor links at the top of the page, instead opting to scroll through all items

In these cases users found the list overwhelming

Some users did not use the “back to top” links, instead opting to scroll to the top of the page

Design decisions

Clearly labelling each alphabetical section helps users orientate themselves. Placing each condition or symptom link in a white panel separated by a 1px divider, gives long lists structure and makes them easier to scan.

Accessibility

The A-Z list is contained within a <nav> element. This element has an aria label attribute with the value Health A-Z Navigation. This will be read out by screen readers that support this attribute to help clarify what the list of links are.
