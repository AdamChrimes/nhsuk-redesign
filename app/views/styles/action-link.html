What it is

The action link is intended to bring attention to onward tasks within a page of content, without resorting to putting a link in a callout box. This is to improve the flow of the page, when there are potentially numerous other components on a page.

What was being used

Priority link used on nhs.uk

Previous research

None available.

User need

As a user I NEED to know which links on the page can take me to important information or useful tools SO THAT I can continue to find important information relevant to my situation.

Design and research

The arrow should be green as this is our action colour.

First iteration

Version 1

Research findings

NHS.UK pop up testing, Leeds, 1st June 2018

  • All users missed the action link, even when prompted when it was in view.
  • “Nothing stands out”
  • “It didn’t jump out”

Second iteration

Hypothesis

If we increase the size of the text to larger than the body text size, and we changed the colour of the icon to green, users will see them more easily.

Earlier versions used NHS blue, but these were not noticed by test participants, so green was identified as an ‘action’ colour, to be used for action links as well as buttons.

Version 2

Research findings

NHS.UK usability lab, Salford, 4th July 2018

  • Most users understood the purpose of the action-link, but some felt unsure what to expect when clicking on it

NHS.UK usability lab, Sheffield, 16th July 2018

  • 3/5 users spontaneously pointed out action links and indicated that they would find this useful.
  • Those who didn’t use them stated they would already know their local A&E or use another method to locate it.

Accessibility

Usage (rules)

  • Only use action links to direct users to the beginning of a task.
  • Do not use action links simply for linking to other pages, or other sites.
  • More than one action link can be used per page (if, for example, two separate paragraphs of text require an action link to guide users to a related task), but do not stack multiple action links in the same place. This lessens the visual impact of the link.
  • The wording of an action link should always begin with a verb, to communicate to the user that they are about to begin a task. For example, ‘Apply for an EHIC card’, or ‘Book an appointment’. Links should be short and only mention a single task.
  • Action links can be used within care cards.
