Code and design

When to use the header

Use this header at the top of every page if your service is part of the new NHS.UK website, replacing what was NHS Choices.

If your service is not part of what was NHS Choices, get in touch on the NHS.UK #standards Slack channel or email service-manual@nhs.net. We are considering how to use the header on other NHS Digital sites and services on the NHS.UK domain including, for example, some transactional journeys.

How to use the header

For smaller viewports, for example on mobile devices, the search and navigation items are hidden and can be expanded by buttons, which need JavaScript to work. You can find the JavaScript code above.

For the search suggestions functionality, you need two JavaScript files:

  • typeahead.bundle.min.js

Research

Our research has shown that this header is clearer and more usable than the old NHS Choices header, and it expresses the NHS brand.

Removing drop downs on larger screens, for example, has made it easier for people with motor skills problems or touchscreens.

Get in touch to share your research findings if you've used this pattern.
